Session statuses (what each means)
  • Open — Run is on the calendar. The site shows a green “Open” badge. Who can pay is controlled only by Drop it / Lock payments below.
  • Completed — Set automatically after the run’s calendar date in Los Angeles (payments turn off). Past runs stay visible in admin; you can reopen as Open if needed.
  • Cancel run — Run is off. It disappears from the public signup flow (same as iOS “Cancelled”).
  • Delete event — Permanently removes the row and its RSVPs (good for test sessions). Card payments are not auto-refunded; use Stripe if someone paid.
